## Setup

```
git clone https://github.com/strichliste/strichliste-web-e2e
cd strichliste-web-e2e
yarn
```

## Use

- During development `yarn start`
- On build server `yarn test`

## Housekeeping

- Update dependencies to latest : `yarn update`
